## Research

I study explainable AI and large language models for prediction and decision-making in manufacturing and production systems. My current work connects rule extraction, natural-language explanations, and human feedback so that models used in the field can be understood and acted on.

- Human-centered XAI and LLM-generated explanations
- Production intelligence for semiconductor manufacturing
- Transformer-based wind-power and time-series forecasting
